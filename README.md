# Sales Web MVC

Project developed by me during a Udemy course with the goal of learning and applying **ASP.NET Core MVC** concepts in the development of a web application.

## 📖 About the Project

In this project, I will develop a small system for managing **sellers**, **departments**, and **sales**.

The objective is to understand in practice how the **MVC (Model-View-Controller)** architecture works and how to structure a web application using **ASP.NET Core**.

During the development of the project, important concepts will be applied, such as:

* MVC Architecture
* Use of the **Razor Template Engine**
* Layout structure using **Bootstrap**
* **CRUD operations** (Create, Read, Update, and Delete)
* Queries and searches using **LINQ**
* Organization of application layers

## 🛠 Technologies Used

* C#
* ASP.NET Core MVC
* Entity Framework Core
* LINQ
* Razor
* Bootstrap

## 📂 Project Structure

The project follows the standard **ASP.NET Core MVC** structure:

* **Models** → Representation of the system entities
* **Views** → User interface built with Razor
* **Controllers** → Responsible for handling requests and application logic
* **Data** → Database context and configuration

## 🎯 Objective

The goal of this project is to strengthen my knowledge in:

* Web development with **ASP.NET Core**
* Structuring applications using the **MVC pattern**
* Data manipulation using **Entity Framework** and **LINQ**
